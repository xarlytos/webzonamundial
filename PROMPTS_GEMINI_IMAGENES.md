## Instrucciones Adicionales para Gemini

- Siempre usar fondos oscuros (#060B14, #0F1D32)
- Acentos dorados (#c9a84c) como color principal de highlight
- Mantener consistencia visual entre todas las imágenes
- Estilo futurista pero elegante, no saturado
- Optimizar para web (compresión sin pérdida de calidad)
- Generar en múltiples resoluciones: 1x, 2x, 3x para retina

## Formato de Salida Deseado
- PNG con transparencia cuando aplique
- JPG para fotografías y fondos
- SVG para iconos y elementos vectoriales
- WebP para optimización web moderna
